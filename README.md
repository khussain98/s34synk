# s34synk
Homemade Addon for Home Assistant. Allows you to integrate your Sunsynk account with the Official API for real-time solar energy monitoring, and enabling smart automations and custom alerts. Stay in control with all the customisations at your fingertips.

V1 Release:
- Current Read Only API
- Fixed SSL Handling


  Configuration:
  -Install Plugin
  -Set Email
  -Set Password
  -Set Poll Interval to 60
  -Press OK
  -Next screen should autofetch a list where you can select your inverter.
  -First Log Run will output all read only settings, which is used later on for writing back to the API.
